PROYECTO CODER 
El siguiente proyecto simula un e-commerce de una libreria
En ella se le permite al usuario seleccionar el genero de libro el cual quiera comprar y filtrar la disponibilidad.
El cliente podra sumar cantidades, alternar entre generos. Ideada principalmente para clientes recurrentes de la lectura, el carrito se encuentra intacto tras refrescar la pagina o volver a abrir la tienda, para que el usuario continue su compra. Este se vacia tanto de manera manual
con el boton de "vaciar carrito" o simplemente al finalizar la compra.
Contiene un apartado para ingresar el DNI, permitiendo asi que aquellos que sean socios tengan acceso a un descuento. En el caso de que se ingrese un dni el cual no se encuentra en la base de socios, simplemente devuelve que no se encuentra registrado.
Por ultimo se calcula el Total de la compra y se devuelve que fue realizada con exito.
